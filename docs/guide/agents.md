# Agents

The Agents section is where you create and manage AI personas (also called Consumer Personas). These agents represent different viewpoints, roles, or demographic profiles for research and analysis.

## Agent Gallery

![Agents Gallery](/screenshots/agents/01-agents-gallery.png)

The main view displays all your agents in a grid layout:

- **Photo cards** - Agents with custom profile images
- **Letter cards** - Agents without images show colored cards with initials
- **Status indicators** - Active, draft, or archived states

## Searching and Filtering

![Agents Search](/screenshots/agents/02-agents-search.png)

### Search
Use the search field to find agents by name quickly.

### Filters

| Filter | Description |
|--------|-------------|
| **Newest** | Sort by creation date |
| **Type** | Filter by agent category (Assistant, Persona, etc.) |
| **Model** | Filter by underlying AI model |
| **Status** | Filter by active/inactive/draft status |

## Agent Types

Vurvey supports several agent types:

| Type | Purpose |
|------|---------|
| **Assistant** | General-purpose AI helpers |
| **Consumer Persona** | Represents specific demographic profiles |
| **Expert** | Specialized knowledge in specific domains |
| **Analyst** | Data analysis and insights generation |

## Viewing Agent Details

Click any agent card to open the detail panel:

![Agent Detail](/screenshots/agents/04-agent-detail.png)

### Detail Panel Contents

- **Profile image** - Visual representation
- **Name** - Agent identifier
- **Type** - Category classification
- **Description** - Full personality and capability description
- **Chat input** - Start a conversation directly

### Example: Miles Thorne

> "Miles Thorne is an AI persona specifically engineered to inject wit and contrarian perspectives into complex discussions. As a Devil's Advocate Agent, he masterfully utilizes debate tactics to challenge assumptions and stress test ideas, ensuring a comprehensive examination of all viewpoints."

## Creating Agents

Click **+ Create Agent** in the top right to build a new AI persona.

### Agent Configuration

1. **Basic Info**
   - Name
   - Avatar/profile image
   - Type selection

2. **Personality**
   - Communication style
   - Tone and voice
   - Character traits

3. **Knowledge**
   - Specialization areas
   - Connected datasets
   - Domain expertise

4. **Instructions**
   - Behavioral guidelines
   - Response formatting
   - Constraints and boundaries

### Best Practices

::: tip Creating Effective Agents
- **Be specific** - Define clear personality traits
- **Add context** - Include background and expertise areas
- **Set boundaries** - Define what the agent should/shouldn't discuss
- **Test thoroughly** - Validate responses match expectations
:::

## Agent Categories

Organize agents by use case:

- **Research** - For conducting surveys and analysis
- **Support** - For customer interaction simulations
- **Creative** - For brainstorming and ideation
- **Technical** - For data analysis and reporting

## Managing Agents

### Edit
- Click an agent card to view details
- Modify any configuration
- Save changes

### Duplicate
- Create copies of existing agents
- Modify for variations
- Test different configurations

### Archive
- Hide inactive agents
- Preserve configuration
- Restore when needed

### Delete
- Permanently remove agents
- Cannot be undone
- Consider archiving instead
