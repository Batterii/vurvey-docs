# Campaigns

Campaigns are structured research initiatives where agents or real participants answer questions and complete tasks.

## Campaign Gallery

![Campaigns Gallery](/screenshots/campaigns/01-campaigns-gallery.png)

The main view shows all campaigns with key information at a glance.

### Campaign Card Information
- **Title** - Campaign name
- **Status** - Draft, Open, or Closed
- **Progress** - Completion percentage
- **Responses** - Number of completes
- **Creator** - Who created the campaign

### Status Indicators

| Status | Meaning |
|--------|---------|
| **Draft** | In development, not yet live |
| **Open** | Actively collecting responses |
| **Closed** | Response collection complete |

## Navigation Tabs

| Tab | Purpose |
|-----|---------|
| **All Campaigns** | View all your campaigns |
| **Templates** | Browse pre-built campaign templates |
| **Usage** | Track credit consumption and metrics |
| **Magic Reels** | Media summaries and highlight reels |

## Filtering and Sorting

- **Search** - Find campaigns by name
- **All statuses** - Filter by Draft/Open/Closed
- **Most Recently Updated** - Sort order options

### Credit Balance
The header shows your current credit balance (e.g., "Current balance: 6,853 credits").

## Best Practices

::: tip Campaign Success
- **Clear objectives** - Know what you want to learn
- **Concise questions** - Respect participant time
- **Logical flow** - Order questions thoughtfully
- **Test first** - Preview before launching
:::

### Question Writing
- Use neutral language
- Avoid leading questions
- Keep language simple
- Provide clear instructions

### Targeting
- Define audience clearly
- Set realistic quotas
- Balance sample size vs. cost
