# Quick Reference

A practical cheat sheet for getting things done in Vurvey. Bookmark this page for fast answers to "where do I find that?" and "how do I do this?"

## Navigation Guide

Use the left sidebar to move between sections:

| Sidebar Item | What You'll Find There |
|---|---|
| **Home** | Your AI chat interface for asking questions, analyzing data, and generating insights |
| **Agents** | Your library of AI personas -- create, customize, and manage specialized AI team members |
| **People** | Audience management -- populations, individual contacts, segments, and custom properties |
| **Campaigns** | Research surveys -- create, launch, monitor, and analyze response data |
| **Datasets** | File library -- upload documents, spreadsheets, media, and other files for AI analysis |
| **Forecast** | Predictive analytics and trend projections based on your research data |
| **Workflow** | Automated research pipelines -- build multi-step AI processes that run on their own *(Beta)* |

## Common Tasks at a Glance

### Getting Answers Fast

| What You Want to Do | Where to Go | Steps |
|---|---|---|
| Ask the AI a quick question | Home | Type your question, press Enter |
| Chat with a specific agent | Home | Click the Agents button in the toolbar, select your agent, then chat |
| Analyze your own data | Home | Click the Sources button, select your campaigns/datasets, then ask |
| Search the web for trends | Home | Click the **Tools** button to enable web research, then ask your question |
| Combine internal data with web research | Home | Enable both **Sources** and **Tools** in the toolbar |
| Get multiple AI perspectives | Home | Type `@AgentName` in your message to bring in additional agents |
| Generate images | Home | Click the **Images** button, then describe what you want |

### Creating and Managing Content

| What You Want to Do | Where to Go | Steps |
|---|---|---|

| Build an audience group | People | Go to Populations tab and create a new group |

| Set up an automated workflow | Workflow | Click **Create new workflow** and design your pipeline |
| Schedule a recurring workflow | Workflow | Open a workflow, click **Schedule** |

## Agent Builder: The 6 Steps

When you create a new agent, you'll walk through these steps:

| Step | Name | What You'll Do | Key Tip |
|---|---|---|---|
| 1 | **Objective** | Choose the agent type and define its mission | Be specific -- "analyze beauty trends for Gen Z" beats "help with research" |
| 2 | **Facets** | Select personality traits that shape how it communicates | Match demographics to your target audience |
| 3 | **Optional Settings** | Add knowledge, tools, datasets, and rules | Connect 1-3 focused datasets for best results |
| 4 | **Identity** | Set the agent's name, biography, and voice | Use the Generate button for AI-suggested names and bios |
| 5 | **Appearance** | Upload or generate an avatar image | Write specific physical descriptions for better AI-generated avatars |
| 6 | **Review** | Test the agent and activate it | Test with 10+ different questions before activating |

|---|---|---|

## AI Model Comparison

| Model | Speed | Best For |
|---|---|---|
| **Gemini Flash** | Fastest | Most everyday tasks, quick analysis, general Q&A |
| **Gemini Pro** | Medium | Deep reasoning, image/video analysis, complex tasks |
| **Claude** | Medium | Staying in character, nuanced instructions, natural writing |
| **GPT-4o** | Medium | Broad general knowledge, creative tasks |

::: tip When in Doubt
Start with **Gemini Flash** for most agents. Switch to **Claude** when persona consistency matters (consumer persona agents), or **Gemini Pro** when the agent needs to analyze visual content.
:::

## Dataset File Statuses

After you upload a file, watch for these status indicators:

| Status | What It Means | Action Needed |
|---|---|---|
| **Uploaded** | File received, waiting to be processed | Wait -- processing starts automatically |
| **Processing** | AI is analyzing the file contents | Wait -- status updates every 30 seconds |
| **Success** | Ready to use in conversations and analysis | None -- start using it! |
| **Failed** | Something went wrong | Click the retry icon next to the file |

## Workflow Statuses

| Status | What It Means | Action Needed |
|---|---|---|
| **Pending** | Queued and waiting to start | Wait for execution to begin |
| **Running** | Currently executing its steps | Monitor progress on the Run tab |
| **Paused** | Temporarily paused | Resume when ready |
| **Completed** | Finished successfully | Review results in Conversations |
| **Failed** | An error occurred during execution | Check the Run tab for error details |

## Tips for Effective AI Prompting

Getting better results from Vurvey's AI starts with how you frame your questions:

### Be Specific
Instead of "Tell me about my data," try "What are the top 3 themes from my Q1 packaging survey, with supporting quotes for each?"

### Request a Format
Ask for tables, bullet points, numbered lists, or executive summaries to get organized output.

### Narrow Your Sources
Select one campaign or dataset at a time for more focused, accurate answers.

### Ask for Evidence
Include "with supporting quotes" or "cite your sources" in your prompt.

### Iterate and Drill Down
Ask follow-up questions to go deeper: "Tell me more about the second theme" or "Break that down by age group."

### Set Constraints
Tell the AI what to include or exclude: "Only use data from female respondents aged 25-34."

### Use Templates
Common prompt patterns for research:

| Task | Prompt Template |
|---|---|
| **Theme extraction** | "Identify the top [N] themes from [source]. For each theme, provide 2-3 supporting quotes." |
| **Executive summary** | "Write a [length] executive summary of [source] suitable for [audience]." |
| **Comparison** | "Compare [source A] vs [source B]. What are the key differences and similarities?" |
| **Sentiment analysis** | "Analyze the sentiment across all responses in [source]. Break down by positive, negative, and neutral." |
| **Recommendation** | "Based on the findings in [source], provide [N] actionable recommendations for [team/goal]." |
| **Trend identification** | "What trends or patterns emerge when looking at [source] over time?" |

## Troubleshooting Quick Fixes

### Chat isn't responding
1. Check your internet connection
2. Refresh the browser page
3. Try a different browser
4. Clear your browser cache and cookies
5. Check if an ad blocker or browser extension is interfering

### File upload failed
1. Make sure the file format is supported (see list above)
2. Check if the file exceeds the size limit
3. Try uploading again
4. If the file is very large, try splitting it into smaller parts
5. Check your internet connection stability

### Campaign won't launch
1. Check that all required fields are filled in
2. Verify your account has sufficient credits
3. Make sure at least one question is added
4. Preview the campaign to spot any issues
5. Look for validation error messages on the page

### Dataset stuck on "Processing"
1. Wait a moment -- processing can take time for large files (up to 30 minutes for video)
2. Refresh the page to get the latest status
3. If a file shows "Failed," click retry
4. Contact support if processing seems stuck for an extended period

### Workflow won't run
1. Check for validation errors in the workflow builder
2. Make sure all changes are saved
3. Verify that agents are assigned to each step
4. Try running it manually first before setting up a schedule
5. Check that all agent task prompts are filled in

### Agent responses seem off
1. Review the agent's mission and rules -- are they specific enough?
2. Check that the right datasets are connected
3. Try a different AI model
4. Test with simpler questions first to isolate the issue
5. Review the agent's facets for conflicting settings

### Can't share with a teammate
1. Make sure they have an account in your workspace
2. Check that you have Share permission on the resource
3. Try searching for their name or email in the share dialog
4. Ask your workspace administrator if they can help

### Search results seem outdated
1. Web search pulls current information -- but results depend on what's available online
2. Try rephrasing your search query
3. Include specific date ranges in your question: "What happened in [topic] in the last 30 days?"

## Glossary

| Term | What It Means |
|---|---|
| **Agent** | An AI persona configured with specific expertise, personality, and knowledge |
| **Population** | A group of synthetic or real participants used in research |
| **Campaign** | A research survey or study that collects responses from participants |
| **Dataset** | A collection of uploaded files that the AI can analyze and reference |
| **Workflow** | An automated multi-step research process using AI agents |
| **Credits** | The currency used for AI operations on the platform |
| **Facets** | The demographic and personality traits that define an agent's character |
| **Magic Summary** | An AI-generated summary of your campaign or dataset |
| **Magic Reels** | AI-compiled video highlights from campaign responses |
| **Workspace** | A shared environment where your team collaborates on research |
| **Mold** | A reusable template for creating consistent AI persona profiles *(Enterprise)* |
| **Segment** | A dynamic audience group that automatically updates based on rules |
| **List** | A static, manually managed group of contacts |
| **Sources** | The campaigns, datasets, and files you connect to an AI conversation |
| **Citation** | A reference showing which source the AI used for a specific claim |
| **Stimulus** | An image, video, or AR model shown to respondents alongside a question |
