# Home (Chat Interface)

The Home page is your primary workspace for interacting with Vurvey's AI capabilities through a conversational interface.

## Overview

![Chat Interface](/screenshots/home/01-chat-main.png)

The chat interface provides a unified way to query agents, attach data, and control how the AI reasons about your requests.

## Key Elements

### Chat Input

The central input field displays **"Ask anything..."** - type your questions or commands here to interact with AI agents.

Features:
- Natural language queries
- Multi-line input support
- File attachments (drag and drop)

### Agent Selector

Click the **Agent** dropdown to choose which AI persona will respond to your queries.

Each agent has different:
- Specializations
- Personalities
- Knowledge bases
- Communication styles

### Sources

Click **Sources** to control which datasets or documents are available to the agent during the conversation.

- Select specific datasets
- Filter by file type
- Search across sources

### Tools

Click **Tools** to enable specialized capabilities:

- Data analysis and visualization
- Export functionality
- Calculation tools
- Custom integrations

## Conversation Sidebar

![Conversation Sidebar](/screenshots/home/04-conversation-sidebar.png)

The left panel manages your conversation history:

### Recent Conversations
- Listed by title or first message
- Click to continue any conversation
- Automatically saved

### Creating New Chats
- Click **+** next to "Conversations"
- Start fresh with selected agent
- Previous context not carried over

### View All
- Click "View all" for complete history
- Search past conversations
- Filter by date or agent

## User Profile

Located at the bottom left:
- Your name and email
- Workspace indicator
- Refresh and logout options

## Tips for Effective Chat

1. **Be specific** - The more context you provide, the better the response
2. **Use the right mode** - Match your query type to the mode
3. **Select appropriate sources** - Focus the AI on relevant data
4. **Iterate** - Ask follow-up questions to refine results

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| Send message | `Enter` |
| New line | `Shift + Enter` |
| New conversation | Click `+` |
