# About This Documentation

This documentation site is maintained alongside the Vurvey platform to ensure it stays accurate and up to date.

## How We Keep It Current

- **Screenshots** are captured regularly from the live platform, so the images you see reflect the current version of Vurvey
- **Content** is reviewed and updated with each platform release to cover new features and changes
- **Links** are automatically validated to ensure nothing is broken

## Something Look Outdated?

If you notice a screenshot that doesn't match what you see in the app, or if a description seems incorrect, please reach out to your Vurvey support contact. Your feedback helps us keep this documentation accurate for everyone.
